# ЮгТрансГаз

Современная версия сайта-визитки для сервиса установки и обслуживания газобаллонного оборудования.

## Структура

- `frontend` - React + Vite + TypeScript
- `backend` - заготовка под Node.js + Express + Prisma + PostgreSQL

## Быстрый старт фронтенда

```bash
npm install
npm run dev
```

Команды из корня проекта:

```bash
npm run dev
npm run build
npm run lint
```
